# ⏳ DadLine  
### AI-Powered Deadline & Productivity Manager

DadLine is an intelligent productivity platform designed to help students, professionals, and teams manage deadlines without overwhelm. It combines smart task prioritisation, adaptive scheduling, and productivity insights into a clean, modern workspace.

Built with **React + Vite**, DadLine focuses on reducing procrastination by turning chaotic task lists into structured execution plans.

---

## 🚀 Core Features

### 🧠 Smart Deadline Prioritisation
Ranks tasks based on:
- urgency
- workload
- estimated effort
- approaching deadlines

### 📅 Adaptive Scheduling
Helps users organise work blocks around:
- due dates
- task priority
- available time
- workload balance

### 🔔 Deadline Reminders
Provides reminders for:
- upcoming deadlines
- overdue tasks
- high-priority work
- focus sessions

### 📊 Productivity Overview
Track:
- completed tasks
- pending tasks
- missed deadlines
- workload distribution

### 🎯 Focus Mode
A minimal distraction interface for deep work sessions.

### 🌙 Modern UI
- responsive dashboard
- clean task layout
- fast Vite-powered frontend
- modern React component structure
